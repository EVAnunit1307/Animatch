# Animatch

Small project skeleton for matching anime images or landmarks.

Quick start

1. Create a virtual environment and activate it:

   python -m venv .venv
   .\.venv\Scripts\activate

2. Install dependencies:

   pip install -r requirements.txt

3. Run the demo entry point:

   python -m animatch.app.main

Project structure

- `animatch/app/` — application code
- `animatch/app/services/` — service modules (landmarks, features, match, explain)
- `animatch/app/data/anime_vectors.json` — sample data file

This is a placeholder skeleton — implement domain logic and tests as needed.
